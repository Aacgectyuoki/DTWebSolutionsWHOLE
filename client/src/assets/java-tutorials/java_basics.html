Java Basics

Java is a high-level, object-oriented programming language known for its platform independence. It was first released by Sun Microsystems in 1995.

Key features of Java:

  • Write Once, Run Anywhere (WORA): Java code can run on any platform with a Java Virtual Machine (JVM).
  • Strongly Typed: Data types are strictly defined in Java.
  • Garbage Collection: Java manages memory automatically.

Variables

Java uses variables to store data. Variables must be declared with a specific data type. For example, you can declare an integer variable as follows:

int myNumber = 42;

Control Structures

Control structures allow you to control the flow of your Java program. For example, the if-else statement is commonly used for conditional execution:

int x = 10;
if (x > 5) {
    System.out.println("x is greater than 5.");
} else {
    System.out.println("x is not greater than 5.");
}

Loops

Loops enable you to repeat a block of code. A common loop in Java is the "for" loop:

for (int i = 1; i <= 5; i++) {
    System.out.println("Iteration " + i);
}

Data Types

Java supports various data types such as int, float, double, and char. You can choose the appropriate data type for your variables based on your needs. For example:

double price = 19.99;
char grade = 'A';

Methods

Methods are functions in Java that perform specific tasks. You can create your own methods or use built-in methods from Java's standard library. Here's an example:

public static int add(int a, int b) {
    return a + b;
}

int result = add(3, 4);

Arrays

Arrays are used to store multiple values in a single variable. For example, an array of integers:

int[] numbers = {1, 2, 3, 4, 5};

Exception Handling

Java provides a robust exception handling mechanism. You can catch and handle exceptions in your code to ensure graceful error recovery. For example:

try {
    int result = 10 / 0; // This will throw an ArithmeticException
} catch (ArithmeticException e) {
    System.out.println("An error occurred: " + e.getMessage());
}

File I/O

Java supports file input and output operations. You can read from and write to files using classes like BufferedReader and FileWriter. Here's an example of reading from a text file:

import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class ReadFileExample {
    public static void main(String[] args) {
        try {
            BufferedReader reader = new BufferedReader(new FileReader("example.txt"));
            String line;
            while ((line = reader.readLine()) != null) {
                System.out.println(line);
            }
            reader.close();
        } catch (IOException e) {
            System.out.println("An error occurred: " + e.getMessage());
        }
    }
}

Collections Framework

Java provides a rich set of data structures in its Collections Framework. You can use ArrayList, HashMap, and other data structures for various purposes in your programs. For example:

import java.util.ArrayList;
import java.util.HashMap;

public class CollectionsExample {
    public static void main(String[] args) {
        // Create an ArrayList of strings
        ArrayList fruits = new ArrayList<>();
        fruits.add("Apple");
        fruits.add("Banana");
        fruits.add("Orange");

        // Create a HashMap of key-value pairs
        HashMap ages = new HashMap<>();
        ages.put("Alice", 30);
        ages.put("Bob", 25);
        ages.put("Charlie", 35);
    }
}

Threads and Concurrency

Java allows multithreading for concurrent execution. You can create and manage multiple threads to perform tasks simultaneously. For example:

class MyThread extends Thread {
    public void run() {
        for (int i = 1; i <= 5; i++) {
            System.out.println("Thread " + getId() + ": " + i);
        }
    }
}

public class MultithreadingExample {
    public static void main(String[] args) {
        MyThread thread1 = new MyThread();
        MyThread thread2 = new MyThread();
        thread1.start();
        thread2.start();
    }
}

Serialization

Java allows you to serialize objects for data storage and transmission. You can use object serialization to save and load objects. For example:

import java.io.*;

class Person implements Serializable {
    private String name;
    private int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
}

public class SerializationExample {
    public static void main(String[] args) {
        Person person = new Person("Alice", 30);
        
        try {
            // Serialize the object
            FileOutputStream fileOut = new FileOutputStream("person.ser");
            ObjectOutputStream out = new ObjectOutputStream(fileOut);
            out.writeObject(person);
            out.close();
            fileOut.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
