Core Java Topics

Introduction to Variables

Variables are used to store data in Java. They must be declared with a data type before use. Here's an example:

int myNumber = 42;
String myText = "Hello, Java!";

Control Structures

Control structures are used to control the flow of a Java program. The if-else statement is a common example:

int x = 10;
if (x > 5) {
    System.out.println("x is greater than 5.");
} else {
    System.out.println("x is not greater than 5.");
}

Loops

Loops allow you to execute a block of code repeatedly. Here's an example using a for loop:

for (int i = 1; i <= 5; i++) {
    System.out.println("Iteration " + i);
}

Arrays

Arrays are used to store multiple values in a single variable. Here's an example of declaring and initializing an array:

int[] numbers = {1, 2, 3, 4, 5};

Methods

Methods are functions that perform specific tasks. Here's an example of defining and calling a method:

public static int add(int a, int b) {
    return a + b;
}

int result = add(3, 4);

Object-Oriented Programming (OOP)

Java is an object-oriented programming language. It supports the creation and use of objects. Here's an example of a simple Java class:

class Person {
    String name;
    int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public void sayHello() {
        System.out.println("Hello, my name is " + name);
    }
}

// Create an instance of the Person class
Person person1 = new Person("John", 30);
person1.sayHello();

Inheritance

Inheritance is a fundamental concept in OOP. It allows a class to inherit properties and methods from another class. Here's an example:

class Animal {
    void speak() {
        System.out.println("Animal speaks");
    }
}

class Dog extends Animal {
    void speak() {
        System.out.println("Dog barks");
    }
}

Dog myDog = new Dog();
myDog.speak(); // Outputs "Dog barks"

Exception Handling

Java provides a robust exception handling mechanism. Here's an example of handling exceptions:

try {
    int result = 10 / 0; // This will throw an ArithmeticException
} catch (ArithmeticException e) {
    System.out.println("An error occurred: " + e.getMessage());
}

File I/O

Java supports file input and output operations. Here's an example of reading from a text file:

import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class ReadFileExample {
    public static void main(String[] args) {
        try {
            BufferedReader reader = new BufferedReader(new FileReader("example.txt"));
            String line;
            while ((line = reader.readLine()) != null) {
                System.out.println(line);
            }
            reader.close();
        } catch (IOException e) {
            System.err.println("An error occurred: " + e.getMessage());
        }
    }
}

Collections Framework

Java provides a rich set of data structures in its Collections Framework. Here's an example of using an ArrayList:

import java.util.ArrayList;
import java.util.List;

public class ArrayListExample {
    public static void main(String[] args) {
        List fruits = new ArrayList<>();
        fruits.add("Apple");
        fruits.add("Banana");
        fruits.add("Orange");
        System.out.println("Fruits: " + fruits);
    }
}

Threads and Concurrency

Java allows multithreading for concurrent execution. Here's an example of creating and running multiple threads:

class MyThread extends Thread {
    public void run() {
        for (int i = 1; i <= 5; i++) {
            System.out.println("Thread " + getId() + ": " + i);
        }
    }
}

public class MultithreadingExample {
    public static void main(String[] args) {
        MyThread thread1 = new MyThread();
        MyThread thread2 = new MyThread();
        thread1.start();
        thread2.start();
    }
}

Serialization

Java allows you to serialize objects for data storage and transmission. Here's an example of object serialization:

import java.io.*;

class Person implements Serializable {
    private String name;
    private int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
}

public class SerializationExample {
    public static void main(String[] args) {
        try {
            Person person = new Person("Alice", 25);
            ObjectOutputStream out = new ObjectOutputStream(new FileOutputStream("person.ser"));
            out.writeObject(person);
            out.close();
        } catch (IOException e) {
            System.err.println("An error occurred: " + e.getMessage());
        }
    }
}

Java Standard Library

Java comes with a vast standard library that simplifies many programming tasks. Here's an example using the `java.util` package:

import java.util.ArrayList;
import java.util.Collections;
import java.util.List;

public class CollectionsExample {
    public static void main(String[] args) {
        List numbers = new ArrayList<>();
        numbers.add(3);
        numbers.add(1);
        numbers.add(2);
        Collections.sort(numbers);
        System.out.println("Sorted numbers: " + numbers);
    }
}
