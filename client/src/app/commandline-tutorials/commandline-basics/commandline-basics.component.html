Command Line Basics

Introduction

The command line, or command-line interface (CLI), is a text-based method of interacting with a computer's operating system that enables users to perform tasks without relying on a mouse. It provides a way to precisely control and automate various functions on a computer or server. In this lesson, we will delve into the essential concepts and fundamental commands required to efficiently navigate and operate within a command-line environment, all accomplished through typing text commands rather than using graphical interfaces and a mouse.

It's important to note that different systems have their own command-line interfaces:

  • Windows: Uses Command Prompt (cmd) and PowerShell.
  • Linux/Unix: Offers a Terminal with various shell options, such as Bash, Zsh, etc.
  • macOS: Provides a Terminal with support for Unix-based shells.

Commands

File System Navigation:

  • pwd: Print the current directory.
  • ls: List files and directories.
  • cd: Change the directory.
  • cd ..: Move up one directory.
  • cd ~: Navigate to the home directory.

Working with Files and Directories:

  • mkdir: Create directories.
  • touch: Create empty files.
  • rm: Remove files or directories.
  • cp: Copy files or directories.
  • mv: Move or rename files or directories.
  • cat: Display file contents.

File Permissions:

  • chmod: Change file permissions.
  • chown: Modify file ownership.

File Content Manipulation:

  • grep: Search for text in files.
  • sed: Edit text using regular expressions.
  • awk: Process text data, often in tabular format.

Process Management:

  • ps: List running processes.
  • top or htop: Monitor processes in real-time.
  • kill: Terminate processes.

Input and Output:

  • stdin, stdout, stderr: Standard input, output, and error streams.
  • Pipelines (|): Pass output from one command to another.

Wildcards:

  • *: Matches any characters in a filename.
  • ?: Matches a single character in a filename.

Command Help and Manuals:

  • man: Display command manuals.
  • --help: Access help information for commands.

History:

  • history or pressing up of the arrow key in the keyboard: Access previous commands.
